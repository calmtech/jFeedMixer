jFeedMixer 利用ガイド

jFeedMixerは複数のRSSフィードを統合し、時系列に並べ替えたものを
ウェブサイト上に埋め込んで表示することができる、jQueryプラグインです。

[バージョン]
1.1

[動作環境]
jQuery 1.12.4 で動作確認を行っています。
Google AJAX Feed API が終了したため、簡易的なRSS取得スクリプトを配置しています。

[導入方法]
http://www.calmtech.net/jfeedmixer
